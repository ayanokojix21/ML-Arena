# ML Arena

## Overview

ML Arena is an open-source ML practice repository where you:

* implement machine learning algorithms
* explore real datasets
* improve models step by step
  

## Task Types

Each problem is divided into four types of tasks:

| Type         | Description                                                       |
| ------------ | ----------------------------------------------------------------- |
| Scratch      | Implement the algorithm from first principles using NumPy         |
| Optimization | Improve an existing implementation through tuning or enhancements |
| Library      | Build a baseline using standard libraries like sklearn            |
| Exploration  | Analyze the dataset and visualize patterns and model behavior     |


## Repository Structure


Each problem folder contains the dataset and notebooks for different task types.


## Getting Started

1. Clone the repository

2. Choose a problem from the `problems` directory

3. Select a task type based on your level:

* Beginner: library or exploration
* Intermediate: scratch
* Advanced: optimization

4. Work on the corresponding notebook:

```
scratch/training.ipynb
library/training.ipynb
exploration/exploration.ipynb
```

5. Create a branch, commit your work, and open a pull request linked to an issue


## Datasets

All datasets are already included in the repository. No additional downloads are required.
