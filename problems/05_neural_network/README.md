# 05 Neural Network

## Overview

In this problem, you will build a simple feed forward neural network and understand how learning happens through layered transformations.


## Dataset

Path: `problems/05_neural_network/dataset.csv`

* Source: UCI Wine Quality
* Rows: 6497
* Columns: 13
* Targets:

  * `quality`
  * `quality_binary`

This is a tabular dataset, allowing you to focus on core neural network concepts without image processing.
